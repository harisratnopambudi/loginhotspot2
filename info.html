Tentang Kami

Kami menyediakan layanan internet hotspot berkualitas dengan berbagai pilihan paket. Tersedia juga koleksi template login page premium untuk kebutuhan bisnis WiFi Anda.

Cara Menggunakan Voucher

  1. Beli voucher dari operator hotspot
  2. Hubungkan ke jaringan WiFi
  3. Buka browser, halaman login akan muncul
  4. Masukkan kode voucher
  5. Klik "Login" dan nikmati internet!

Edit Template

Setelah membeli template, Anda akan mendapat file HTML yang bisa diedit di Notepad. Ganti teks, warna, dan gambar sesuai brand Anda, lalu upload ke MikroTik via Winbox > Files.
